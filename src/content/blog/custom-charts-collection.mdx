Visual storytelling is at the heart of Kinetix. We realized that standard chart libraries didn't fit our "sketchbook" aesthetic, so we built our own set of **Astro components** that render directly from our markdown.

This allows us to drop in charts as easily as images, but with full interactivity and CSS styling.

## 1. Bar Charts

Use the `<BarChart />` component to compare values. It supports both a corporate "clean" style and our signature "hand-drawn" look.

### Clean Style
Perfect for formal reports or precise data.

<BarChart 
  data={[
    { label: 'A', value: 10 },
    { label: 'B', value: 25 },
    { label: 'C', value: 15 }
  ]} 
  variant="clean"
  color="#3b82f6" 
/>

### Hand-Drawn Style
Using SVG filters to simulate rough paper edges.

<BarChart 
  data={[
    { label: 'Sketch', value: 80 },
    { label: 'Draw', value: 60 }
  ]} 
  variant="hand-drawn"
  color="#ef4444" 
/>
